## What is Cinemusic?

Cinemusic is an interactive web platform where you can explore, discover, and play music inspired by cinema and more. Built with Angular 19, Cinemusic delivers an immersive experience through a modern interface, intuitive navigation, and comprehensive multilingual support.

<Info>
  Cinemusic is live at [juanpabloleonf.github.io/cinemusic](https://juanpabloleonf.github.io/cinemusic/)
</Info>

## What you can do

With Cinemusic, you can:

- Browse different music sections including albums, playlists, and categories
- Create and manage custom playlists
- Play songs with an advanced music player
- Explore series and content organized by genre
- Switch between Spanish and English interfaces
- Enjoy smooth animations and transitions throughout the app

Cinemusic is built with modern web technologies:

- **Angular 19.1** - Latest version of the Angular framework
- **TypeScript 5.7** - Type-safe JavaScript development
- **RxJS 7.8** - Reactive programming with observables
- **ngx-translate** - Internationalization and localization
- **Karma & Jasmine** - Comprehensive testing framework
- **HTML5 & CSS3** - Modern web standards

## Architecture overview

The application follows a clean, domain-driven architecture:

```
src/app/
├── domain/           # Business logic and models
│   ├── models/       # Domain models (music, series)
│   ├── services/     # Business services
│   ├── states/       # State management
│   ├── interceptors/ # HTTP interceptors
│   └── mappers/      # Data transformation
└── ui/               # Presentation layer
    ├── pages/        # Feature pages (music, series)
    ├── components/   # Reusable UI components
    └── animations/   # Animation definitions
```

This separation ensures maintainability, testability, and scalability as your application grows.
